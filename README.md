# projeto-estrutura-de-dados-ufla

Projeto prático feito durante a disciplina de estrutura de dados na ufla 2023-1

## Como executar o projeto no linux

- se não tiver, instale o `g++`:

```bash
sudo apt install g++
```

- rode `make`:

```bash
make
```

- execute o executavel criado:

```bash
./programa
```
